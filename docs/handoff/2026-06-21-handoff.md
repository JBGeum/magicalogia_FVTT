## 4. 함정 / 참고

- **커밋 메시지**: 영어 한 줄, co-author 없음. lint-staged(prettier/eslint)가 커밋 시 md·scss·hbs·mjs 자동 정렬(정상).
- **시안 토큰 이식 규칙**: 새 핸드오프 CSS의 `--mg-*` 토큰/`.theme-*` 블록은 이식 금지(우리 `scss/theme/_tokens.scss` 재사용). 컴포넌트 규칙만 `.magicalogia { }` 중첩 SCSS로 이식. 신규 색 토큰(예 `--mg-bad*`)은 다크+라이트 **양쪽** 추가해야 `theme-tokens.test.mjs` 완전성 통과.
- **데이터 모델 유지 + 템플릿 재바인딩**(마이그레이션 없음) — 시안 `system.*` 경로는 예시, 현재 모델로 매핑.
- **테마**: 시트는 `game.settings`(client) "theme"로 다크/라이트. 채팅 카드는 **라이트 고정**(메시지 content라 보는 사람별 전환 불가 — 추후 공통 테마 재설계).
- **에디터**: HTMLField는 V13 `<prose-mirror name=… value=… toggled="false">{{{enriched}}}</prose-mirror>`. `{{editor}}`(V12) 쓰지 말 것.
- **partial 참조는 전체 경로**(`{{> "systems/magicalogia/templates/…"}}`) — 짧은 이름 금지(`template-partials.test.mjs`가 차단).
- **아이템 체크박스**: `.mg-check` span + `data-action="toggle-field" data-field="system.…"`(item-sheet `#onToggleField`).
- **Write로 md 생성 시 파일 끝 도구 태그 새어듦** 주의 — `tail`로 확인(이번 세션 prettier가 처리).
- **서버 재시작 vs F5**: `static TABS`/`documentTypes`/settings 등록 변경은 서버 재시작 필요할 수 있음. 템플릿/SCSS만이면 F5.
- **SDD 진행 로그(복구용)**: `.git/sdd/progress.md` — S4·S5·아이템·채팅카드 task별 커밋·리뷰 결과·이월 Minor 전부 기록됨. (`.superpowers/sdd/`는 brief/report/diff 스크래치 — gitignore.)
