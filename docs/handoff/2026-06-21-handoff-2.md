## 4. 함정 / 참고

- **커밋 메시지**: 영어 한 줄 conventional, **co-author 없음**. lint-staged(prettier/eslint)가 커밋 시 md·scss·hbs·mjs 자동 정렬(정상) — md 코드블록도 prettier가 재포맷.
- **시안 토큰 이식 규칙**: 시안 CSS의 `--mg-*`/`.theme-*`는 이식 금지(우리 `scss/theme/_tokens.scss` 재사용). 컴포넌트 규칙만 `.magicalogia { }` 중첩 SCSS로. **신규 색 토큰은 다크+라이트 양쪽 추가**해야 `theme-tokens.test.mjs` 완전성 통과(이번에 `--mg-good*` 추가).
- **데이터 모델 최소 변경 + 템플릿 재바인딩**. 이번 세션 모델 변경은 spell `tn`(수동 폴백 TN) **한 필드뿐**. 시안 `system.*` 경로는 예시 — 현재 모델로 매핑.
- **채팅 카드는 라이트 고정**(메시지 content라 보는 이별 전환 불가). 시트는 `game.settings`(client) "theme"로 다크/라이트, `applyTheme(element)`로 적용.
- **ApplicationV2**: 시트는 `ActorSheetV2`/`ItemSheetV2`. 신규 standalone 앱은 `SpecialtyPickerApp`(`foundry.applications.api.ApplicationV2` + `HandlebarsApplicationMixin`) 참고 — DEFAULT_OPTIONS.actions에 등록한 `data-action`만 동작, `classes:["magicalogia"]` + `applyTheme(this.element)`로 테마.
- **partial/템플릿 참조는 전체 경로**(`{{> "systems/magicalogia/templates/…"}}`), 신규 템플릿은 `helpers/templates.mjs` `loadTemplates`에 등록(`template-partials.test.mjs`가 차단).
- **순수 로직은 Foundry 비의존 모듈로 분리해 단위 테스트**(`findSpecialtyCoord`/`resolveSpecialtyTn`/`formatCost`/`specialtyGrid`/`classifyRoll`/`renderDie`). Foundry 의존(`castSpell`/시트/앱 렌더)은 F5 육안.
- **차트 특기명 66개는 모두 고유**(이름→좌표 역매핑 `findSpecialtyCoord` 전제) — `specialty-table.test.mjs`가 가드.
- **Write로 md 생성 시 파일 끝 도구 태그 새어듦** 주의 — `tail`로 확인.
- **서버 재시작 vs F5**: 모델/`static TABS`/액션 등록/settings 변경은 서버 재시작 필요할 수 있음. 템플릿/SCSS만이면 F5.
- **SDD 진행 로그(복구용)**: `.git/sdd/progress.md` — 이번 3개 기능 task별 커밋·리뷰·이월 Minor 전부 기록. (`.superpowers/sdd/`는 brief/report/diff 스크래치 — gitignore, `git clean -fdx` 시 소멸.)
