# Changelog

## 1.1.0 - 2026-06-29

캐릭터 시트 능력치/특기표 상호작용 강화.

### 능력치

- 공격/방어/근원/마력/임시마력 수치에 hover ±1 증감 오버레이(헤더 라벨 위, 직접 입력 유지)
- 마력·임시마력 증감 시 compact 챗카드 발행(0.8초 연타 합산)

### 특기표

- 특기 불운 마킹 — 불운 특기 판정 시 굴림 합 −1, 챗카드에 표기
- 습득 편집 모드(자물쇠 토글) — 특기명 클릭으로 습득(보유) 토글, 평소엔 굴림
- 관계 상흔의 침식 영역을 anchor 시트에서 지정 → 특기표 해당 열 강조
- 관계 목록에 메모 컬럼 추가(anchor 시트에서 입력)

## 1.0.0 - 2026-06-28

마기카로기아 시스템 첫 정식 릴리스. 주요 기능:

### 마법전

- 다이스 대결(1:1 상쇄) 판정, 부스트, 집중 방어, MP 대미지 적용
- 입회인(witness) 참여·합산, GM 패널·소켓 오케스트레이션

### 사역마 / 소환

- 사역마 액터·시트, 토큰 소환, 호버 HUD(블록 HP·부스트·특성)
- GM 위임 원격 소환 및 소환별 액터 정리

### 주문 / 장서

- 장서(library) 컴펜디움·주문 시트, 차지(charge) 소비·표시
- 특기 선택기, 병기(ruby) 필드와 적층 챗카드

### 표 / 판정

- 툴바에서 RollTable 굴림(월드→컴펜디움 폴백), 가변 특기 판정
- 마법 차트 가로 배치 토글

### 원형(archetype)

- 원형 액터 시트 및 컴펜디움 등록

### 챗카드 / 인프라

- /roll·roll-table 테마 카드, 차지 변경 카드
- LevelDB 컴펜디움 팩 빌드(소스 JSON 비커밋, 예제 데이터로 릴리스)
- GitHub Actions CI(lint·typecheck·test·build) 및 릴리스 자동화 워크플로
